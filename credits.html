Design Inspiration

The visual palette of Roman Solar Time is inspired by the restored medieval colouration of the Prague Astronomical Clock (Orloj). We gratefully acknowledge this historical marvel as a reference for aesthetic design and solar symbolism.

Solar Calculations

Sunrise and sunset times are computed using the Solar Swift library by Chris Hulbert, which implements the US Naval Observatory’s solar position algorithm. The library is open-source and licensed under the MIT License.

View MIT License (Solar Swift Library)

Development Tools

Roman Solar Time was developed using Xcode and SwiftUI, with its companion website built using standard web tools. Both the app’s diagnostic form and the website’s contact form use Formspree to facilitate optional submissions.
